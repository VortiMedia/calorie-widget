Calorie Calculator Widget

A modern, customizable calorie calculator widget that can be embedded on any website, including Framer sites.

Default Theme (with Lead Capture)
<div id="calorie-calculator"></div> <script src="https://your-domain.com/calorie-widget.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { if (typeof CalorieWidget !== 'undefined') { CalorieWidget.init('calorie-calculator', { requireLeadCapture: true, onLeadCapture: function(data) { console.log('Lead captured:', data); } }); } }); </script>
Default Theme (without Lead Capture)
<div id="calorie-calculator"></div> <script src="https://your-domain.com/calorie-widget.js"></script> <script> document.addEventListener('DOMContentLoaded', function() { if (typeof CalorieWidget !== 'undefined') { CalorieWidget.init('calorie-calculator', { requireLeadCapture: false }); } }); </script>
